# 📝 Blog Application (Work in Progress)
A full-stack blog application where users can create, read, update, and delete posts. This project is being built to strengthen backend development and CRUD operations.

## 🚧 Status
Currently under development and being actively improved.

## ✨ Current Features
* Create blog posts
* View all posts
* Basic backend API setup

## 🔜 Upcoming Features
* User authentication (login/signup)
* Edit and delete posts
* User-specific posts
* Improved UI

## 🛠️ Tech Stack
**Frontend**
* HTML
* JavaScript
* CSS
* EJS(Templating)
  
**Backend**
* Node.js
* Express.js

## 📌 Project Goals
* Learn full-stack development
* Understand CRUD operations
* Build structured backend APIs

## ⚙️ Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Start the server using `npm start`
